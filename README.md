## 🧠 Overview  

This project is an AI-driven Fraud Detection System that analyzes financial transactions in real-time and classifies them as:

- ✅ **Approved** — Safe transaction  
- ⚠️ **Flagged** — Suspicious activity  
- 🚫 **Blocked** — Potential fraud  

Unlike basic systems, this solution focuses on:

- Smart decision-making  
- Clear explanations  
- Real-time interaction  

---

## 🚨 Problem Statement  

Modern fraud detection systems often fail because they are:

- ❌ Too rigid (fixed rules)  
- ❌ Not transparent  
- ❌ Unable to handle complex patterns  

Real-world fraud detection requires:

- Multiple signals  
- Context awareness  
- Clear reasoning  

👉 This project solves that by combining **logic + explainability + simulation**.

---

## ⚙️ How The System Works  

1. User selects a dataset (easy / medium / hard)  
2. Transactions are loaded  
3. Each transaction is processed step-by-step  

The system:

- 📊 Calculates a risk score  
- 🎯 Makes a decision  
- 🧾 Provides a reason  
- 💰 Assigns a reward  

👉 A final report is generated at the end.

---

## 🧠 Core Components  

### 🔹 Risk Scoring  

Each transaction is evaluated using:

- 💰 Amount  
- 👤 User risk level  
- 🔁 Transaction frequency  
- 🌍 Location  
- ⏰ Time  

👉 These factors combine into a **risk score (0 to 1)**
