## 🧠 Overview

**Fraud-AI** is an intelligent, real-time **AI-powered Fraud Detection System** that analyzes financial transactions and classifies them into three categories:

- ✅ **Approved** — Safe and legitimate transaction  
- ⚠️ **Flagged** — Suspicious activity requiring review  
- 🚫 **Blocked** — High probability of fraud

Going beyond traditional rule-based systems, this solution emphasizes **smart decision-making**, **full transparency**, and **interactive real-time processing**.

---

## 🚨 Problem Statement

Traditional fraud detection systems often fall short because they are:
- ❌ Too rigid with fixed rules  
- ❌ Lack transparency (black-box decisions)  
- ❌ Unable to adapt to complex or evolving fraud patterns  

Real-world fraud detection needs **multiple contextual signals**, **smart reasoning**, and **clear explanations**.  

**Fraud-AI** solves this by combining **logic + explainability + simulation**.

---

## ⚙️ How The System Works

1. Select a dataset difficulty (**Easy / Medium / Hard**)
2. Transactions are loaded automatically
3. Each transaction is analyzed step-by-step

For every transaction, the system:
- 📊 Calculates a **Risk Score** (0.0 to 1.0)
- 🎯 Makes a clear **Decision** (Approve / Flag / Block)
- 🧾 Provides a **natural language explanation**
- 💰 Assigns a **performance reward**

A comprehensive **final report** is generated at the end.

---

## 🧠 Core Components

### 🔹 Risk Scoring
Each transaction is evaluated using multiple factors:
- 💰 Transaction Amount
- 👤 User Risk Level
- 🔁 Transaction Frequency
- 🌍 Location (domestic vs foreign)
- ⏰ Time of Day

These inputs are combined into a single **Risk Score** between 0 and 1.

### 🔹 Decision System

| Risk Level | Decision      | Indicator |
|------------|---------------|---------|
| Low        | ✅ Approve    | 🟢      |
| Medium     | ⚠️ Flag       | 🟡      |
| High       | 🚫 Block      | 🔴      |

### 🔹 Explainable AI (XAI)
Every decision is fully transparent.  
**Example:**  
> *Blocked due to high amount + foreign transaction + unusual timing.*

No black-box decisions — you always know **why** the AI made a particular choice.

### 🔹 Performance Metrics
- 📊 Total Transactions Processed
- 🚫 Frauds Detected
- 🎯 Accuracy Rate
- 💰 Total Reward Earned
